# Laravel: Construindo uma API do Zero
Projeto Laravel: construindo uma API do zero, fork do projeto danbonattis/dio-laravel-api

1. Instalar o PHP (7.1 para esse projeto)
2. Instalar o Composer

Após clonar o projeto, rodar na pasta:
```
composer install
```

Para subir um servidor local:
```
php artisan server
```

Saber as rotas disponíveis:
```
php artisan route:list
```
